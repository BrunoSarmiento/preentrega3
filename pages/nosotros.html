US UrbanSport

US UrbanSport es un sitio web realizado a modo de proyecto final del curso "Desarrollo web" dictado por CoderHouse. El sitio representa un catalogo de una tienda de zapatillas, contiene 5 páginas que son: Inicio, Catalogo, Nosotros, Contacto, y los articulos, cada uno con su respectiva página. Para realizar el sitio web se utilizo HTML, CSS, bootstrap, SASS, git y github para controlar las versiones.

Hay 3 versiones del proyecto, la primera donde se empezaba a plantear la idea, intentando encontrar cierto orden en el sitio, sin especificar articulos o nombres de los mismos.

La segunda version ya especificaba nombre de los articulos pero no se diferenciaban entre si, se empezo a pensar un estilo que darle, utilice bootstrap para agregar un header, un footer, y algunas cards para mostrar los articulos. Se agrego un carrusel a la pagina index, el cual mostraba las marcas de los articulos a la venta.

La tercer version del proyeto, ya presenta una idea clara, con 5 secciones bien definidas, una paleta de colores elegida, y todas las paginas del sitio creadas. En esta etapa se utilizo SASS para crear una hoja de estilo legible y ordenada. Tambien se cargo el sitio a github.
