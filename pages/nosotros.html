US UrbanSport

US UrbanSport es un sitio web realizado a modo de proyecto final del curso "Desarrollo web" dictado por CoderHouse. El sitio reresenta un catalogo de una tienda de zapatillas, contiene 5 páginas que son: Inicio, Catalogo, Nosotros, Contacto, y los articulos, cada uno con su respectiva página. Para realizar el sitio web se utilizo HTL, CSS, bootstrap, SASS, git y github para controlar las versiones.
